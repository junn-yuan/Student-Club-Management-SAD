#**APU Student Club Management System**

##**Project Overview**

A comprehensive management solution designed to streamline student club operations at **Asia Pacific University (APU)**. This project follows the full **System Development Life Cycle (SDLC)**, focusing on requirement gathering, system modeling, and high-fidelity prototyping.

-**Status**: Completed (Aug 2025)

-**Prototype Link:** https://m-8594969-53656.bubbleapps.io/version-test?debug_mode=true (Hosted by Project Team)

-**Documentation**: Software Requirements Specification (SRS)

##**My Role: Group Leader & Lead System Analyst**

As the **Group Leader** for a team of 6, I was responsible for the end-to-end coordination of the project. My primary focus was ensuring technical consistency across all modules and managing the structural integrity of our documentation.

**Key Leadership Contributions:**

-**Project Planning:** Established milestones and deadlines using Gantt Charts to ensure timely submission.

-**Requirement Engineering:** Spearheaded the elicitation phase to define functional and non-functional requirements for the SRS.

-**Architecture Design:** Designed the core UML Diagrams (Use Case, Sequence, and Activity) to map out user journeys for the Club Admin and Members.

-**Quality Assurance:** Conducted final reviews of all diagrams and documentation to ensure compliance with industry standards.


#**Tech Stack & Tools**

-**Prototyping:** Bubble.io (No-code web development)

-**Modeling:** Lucidchart / Draw.io (UML, DFD)

-**Documentation:** Microsoft Word (SRS, Project Plan)

-**Management:** Microsoft Project / Gantt Charts

#**Key Deliverables Included in this Repo:**

1. **System Blueprint:** Detailed Use Case diagrams for membership registration and event booking.

2. **Logic Flow:** Activity and Sequence diagrams showing the approval process for new club events.

3. **SRS Document:** A professional-grade 115+ page report covering every technical detail of the system.

4. **UI/UX Wireframes:** Visual guides for the Admin Dashboard and Member Portal.
